# 页面详细说明

## 1. 首页（home）

### 1.1 页面目标
成为高频记录入口，让家长在最短路径下记录孩子问题。

### 1.2 页面模块
1. 顶部欢迎区
2. 快速记录区
3. 今日统计区
4. 最近问题列表

### 1.3 组件结构
- 页面标题：`今天她又问了什么？`
- 录音按钮（主入口）
- 文字输入框
- 拍照按钮
- 今日问题数卡片
- 最近问题列表（最近 3 条）

### 1.4 默认状态
- 显示录音按钮
- 显示空输入框
- 若今日无问题，显示“今天还没有记录问题”

### 1.5 加载状态
- 最近问题列表 skeleton
- 今日统计骨架

### 1.6 空状态
- 空状态文案：`把今天第一个“为什么”记下来吧`
- 可见主按钮：开始记录

### 1.7 错误状态
- 加载失败时显示“加载失败，点击重试”

### 1.8 交互
- 点击录音按钮：进入录音态
- 录音结束后：自动提交或进入确认弹层
- 文字输入提交：生成问题
- 拍照后：允许补充一句原话

---

## 2. 问题详情页（question-detail）

### 2.1 页面目标
展示完整问题卡片，是最核心的内容页。

### 2.2 页面模块
1. 问题头部
2. 原始提问信息
3. 三层回答区
4. 插画区
5. 朗读区
6. 操作区

### 2.3 展示顺序
1. 标准化问题标题
2. 孩子原话
3. 时间 / 分类 / 场景标签
4. 小朋友版回答（默认展开）
5. 故事版回答
6. 家长提示版（默认折叠）
7. 插画
8. 朗读按钮
9. 收藏 / 重新生成 / 删除

### 2.4 默认状态
- 小朋友版默认展开
- 故事版可切换
- 家长提示默认折叠
- 若图片尚未生成，显示占位图与“正在生成插画”

### 2.5 加载状态
- 答案 skeleton
- 图片占位骨架

### 2.6 错误状态
- 回答生成失败：显示失败状态 + 重新生成按钮
- 图片生成失败：显示占位图 + 重新生成图片按钮

### 2.7 交互规则
- 点击“读给孩子听”：默认朗读小朋友版
- 切换到故事版后，朗读故事版
- 收藏按钮应即时反馈
- 删除需要二次确认

---

## 3. 历史页（history）

### 3.1 页面目标
支持长期回顾和搜索。

### 3.2 页面模块
1. 搜索框
2. 分类筛选栏
3. 时间轴列表
4. 收藏筛选入口

### 3.3 默认状态
- 默认按时间倒序展示
- 默认显示全部分类

### 3.4 搜索规则
支持按：
- 关键词
- 分类
- 收藏状态
- 日期范围（一期可先不做复杂日期选择，只保留最近 / 全部）

### 3.5 列表项展示字段
- 标准化问题标题
- 孩子原话简版
- 时间
- 分类
- 收藏状态

### 3.6 空状态
- 无搜索结果：`没有找到相关问题`
- 无历史数据：`把第一个问题记下来，后面就会在这里看到`

### 3.7 错误状态
- 列表拉取失败时显示重试

---

## 4. 周报页（weekly-report）

### 4.1 页面目标
展示成长沉淀价值，帮助家长从更长视角看孩子关注点。

### 4.2 页面模块
1. 本周问题总数
2. 热门主题
3. 重复问题
4. 精选问题
5. 家长建议

### 4.3 默认状态
- 自动显示本周周报
- 如果本周数据不足，展示轻量版提示

### 4.4 空状态
- 文案：`这一周记录得还不多，再多记几个问题就能看到周报了`

### 4.5 错误状态
- 周报生成失败：显示“暂时无法生成周报，请稍后重试”

### 4.6 展示规则
- 热门主题显示 Top3
- 精选问题显示 1-3 条
- 家长建议不超过 3 条

---

## 5. 通用页面与组件规范

### 5.1 通用按钮
- 主按钮：圆角、明显、温和色调
- 次按钮：轻量描边
- 危险按钮：仅删除时使用

### 5.2 文案原则
- 儿童向文案：柔和、短句
- 家长向文案：直接、明确
- 不出现技术术语暴露给用户

### 5.3 空状态优先级
每个页面都必须实现：
- 初始空状态
- 无结果空状态
- 加载中
- 错误态

### 5.4 页面跳转关系
- 首页 → 问题详情页
- 首页 → 历史页
- 首页 → 周报页
- 历史页 → 问题详情页
- 周报页问题卡片 → 问题详情页
