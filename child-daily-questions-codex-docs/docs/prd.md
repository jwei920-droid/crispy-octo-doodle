# 儿童每日疑问小程序 一期 PRD

## 1. 项目概述

### 1.1 项目名称
儿童每日疑问

### 1.2 一句话定义
一款面向家庭自用场景的微信小程序，用于记录孩子每天提出的问题，自动整理为可回看的问题卡片，并生成适合低龄儿童理解的回答与配图。

### 1.3 一期定位
一期为家庭版 MVP，优先验证：
- 家长是否能高频记录孩子问题
- 产品是否能帮助家长“讲给孩子听”
- 问题是否能沉淀成长期可回看的成长记录

### 1.4 一期边界
#### 要做
- 问题记录：语音 / 文字 / 图片
- 自动整理问题卡片
- 三层回答
- 儿童插画
- 语音朗读
- 历史列表、搜索、收藏
- 每周总结

#### 不做
- 自制视频 / 动画
- 搜索或播放外部视频 / 动画
- 家庭协作
- 社交与分享
- 商业化付费

## 2. 用户与场景

### 2.1 目标用户
- 3-6 岁孩子的家长
- 一期核心用户为单个家庭中的爸爸或妈妈

### 2.2 核心场景
1. **即时记录**：孩子路上突然提问，家长先记录
2. **当场解释**：家长需要一版快速、儿童化答案
3. **晚上回顾**：查看当天问题，挑 1-2 个延展讲解
4. **长期沉淀**：观察孩子持续关注的主题

## 3. 产品目标
### 3.1 用户目标
- 快速记录问题，不遗漏
- 获得适龄解释方式
- 能搜索和回看历史问题
- 能看出孩子近期兴趣变化

### 3.2 一期成功指标（建议）
- 首周记录 ≥ 10 个问题
- 单次记录到结果时间可接受
- 历史回看使用率明显
- 每周总结具备实际回看价值

## 4. 产品原则
- **记录优先**：减少输入负担
- **儿童表达优先**：讲给 4-5 岁孩子听
- **家长辅助优先**：帮助家长解释，不替代家长
- **沉淀优先于炫技**：一期先把记录与回看做好

## 5. 核心功能

### 5.1 问题记录
支持：
- 语音输入
- 文字输入
- 图片输入
- 家长备注

记录字段：
- 孩子原话
- 标准化问题
- 时间
- 场景标签
- 输入方式
- 是否收藏

### 5.2 自动整理
自动完成：
- 标准化问题标题
- 分类标签
- 重复问题识别
- 适龄标记

分类建议：
- 动物
- 植物
- 天气
- 身体
- 食物
- 交通
- 自然现象
- 太空宇宙
- 日常生活
- 情绪与社交
- 安全与规则
- 节日与文化

### 5.3 儿童化回答
每个问题输出：
1. **小朋友版**：2-4 句话
2. **故事版**：比喻 / 拟人 / 小故事
3. **家长提示版**：更准确解释 + 延伸追问建议

### 5.4 图片辅助解释
- 每个问题默认一张儿童插画
- 风格统一：绘本感、柔和、友好

### 5.5 语音朗读
- 默认朗读小朋友版
- 支持切换故事版

### 5.6 历史沉淀与搜索
- 历史列表
- 按主题筛选
- 关键词搜索
- 收藏

### 5.7 每周总结
- 本周提问总数
- 热门主题 Top3
- 重复问题
- 精选问题摘要
- 家长建议

## 6. 信息架构
### 页面
1. 首页
2. 问题详情页
3. 历史页
4. 周报页

## 7. 核心流程
### 7.1 主流程
首页记录问题 → 系统整理 → 生成回答/插画 → 查看详情 → 收藏/回看

### 7.2 夜间回顾
历史页查看当天问题 → 进入详情 → 播放故事版或读家长提示

### 7.3 周报
系统汇总每周数据 → 用户进入周报页查看

## 8. 风险
- 儿童化表达过于成人化
- 输入流程过重导致放弃使用
- 配图风格不统一
- 科学问题被讲得过度绝对

## 9. 优先级
### P0
- 首页记录入口
- 问题详情
- 历史列表
- 收藏
- 搜索
- 三层回答

### P1
- 图片生成
- 语音朗读
- 每周总结
- 重复问题识别

### P2
- 回答风格切换
- 导出成长记录
- 知识地图
