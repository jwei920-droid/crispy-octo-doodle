# 设计风格说明（v0.1）

## 1. 总体气质
- 温和
- 安静
- 儿童友好
- 不幼稚化过度
- 适合家长和孩子一起看

## 2. 视觉关键词
- 绘本感
- 柔和留白
- 圆角
- 轻卡片
- 少而明确的颜色层级

## 3. 配色建议
- 主色：柔和暖黄 / 杏色系
- 辅色：浅蓝、浅绿
- 强调色：橙色用于主行动按钮
- 危险色：仅删除时使用，低饱和红

## 4. 字体层级建议
- 页面标题：较大、圆润、清晰
- 模块标题：中号加粗
- 正文：可读性优先
- 次要说明：浅色但保证可读

## 5. 卡片风格
- 白底或浅暖底
- 圆角
- 轻阴影或描边
- 不使用重装饰

## 6. 插画风格
- 儿童绘本风
- 柔和配色
- 角色与物体友好
- 避免写实、避免恐怖感

## 7. 图标风格
- 简单线性或轻填充
- 不使用复杂拟物
- 图标统一圆润感

## 8. 动效原则
- 轻微反馈即可
- 不做复杂炫技动画
- 重点提升点击和状态反馈

## 9. 页面节奏
- 首页强调“快速记录”
- 详情页强调“清晰讲解”
- 历史页强调“好找、好回看”
- 周报页强调“成长沉淀”
