# 验收标准（一期 MVP）

## 1. 总体验收目标
一期应达到：**可记录、可查看、可搜索、可收藏、可生成儿童化回答、可形成周报**。

---

## 2. 首页验收
### 2.1 记录能力
- [ ] 用户可以从首页进行语音输入
- [ ] 用户可以从首页进行文字输入
- [ ] 用户可以从首页进行图片输入
- [ ] 输入后可以创建一条问题记录

### 2.2 页面状态
- [ ] 无数据时有空状态
- [ ] 加载中有 loading 状态
- [ ] 加载失败有错误状态并可重试

---

## 3. 问题详情页验收
### 3.1 基础展示
- [ ] 能展示标准化问题标题
- [ ] 能展示孩子原话
- [ ] 能展示时间与分类
- [ ] 默认展示小朋友版回答
- [ ] 能切换查看故事版
- [ ] 能查看家长提示版

### 3.2 插画与音频
- [ ] 插画加载成功时正常展示
- [ ] 插画失败时可重试
- [ ] 可点击朗读按钮播放内容

### 3.3 操作
- [ ] 收藏按钮有效
- [ ] 删除需二次确认
- [ ] 删除后列表同步变化

---

## 4. 历史页验收
- [ ] 能展示历史问题列表
- [ ] 默认按时间倒序展示
- [ ] 可按关键词搜索
- [ ] 可按分类筛选
- [ ] 可查看收藏问题
- [ ] 空结果时有明确提示

---

## 5. 周报页验收
- [ ] 可显示本周问题总数
- [ ] 可显示热门主题 Top3
- [ ] 可显示重复问题
- [ ] 可显示精选问题
- [ ] 可显示家长建议
- [ ] 数据不足时有轻量提示

---

## 6. AI 内容验收
### 6.1 小朋友版
- [ ] 回答长度适中
- [ ] 语言适合 4-5 岁儿童
- [ ] 不出现明显成人化术语
- [ ] 不惊悚、不恐吓

### 6.2 故事版
- [ ] 有比喻或画面感
- [ ] 不冗长
- [ ] 无明显科学错误

### 6.3 家长提示版
- [ ] 能帮助家长继续解释
- [ ] 至少有 1 个延伸问题建议
- [ ] 不写成学术论文

---

## 7. 稳定性验收
- [ ] 录音失败时有清晰提示
- [ ] 图片上传失败时有清晰提示
- [ ] 回答生成失败时可重新生成
- [ ] 图片生成失败时可重新生成
- [ ] 网络异常不导致页面卡死

---

## 8. 范围验收
### 一期内不得出现
- [ ] 外部视频 / 动画播放
- [ ] 支付功能
- [ ] 社区或社交功能
- [ ] 多用户复杂体系
- [ ] 排行榜 / 积分 / 任务体系

---

## 9. 交付验收
- [ ] 代码结构与文档一致
- [ ] 命名基本统一
- [ ] 页面路由清晰
- [ ] 服务层与页面层职责分明
- [ ] 能基于当前结构继续迭代二期
