## 项目目标
这是一个“儿童每日疑问”微信小程序，一期只做家庭自用 MVP，**不做**以下内容：
- 动画/视频生成
- 外部视频搜索
- 社交
- 支付

一期核心能力是：
1. 记录孩子的问题（语音 / 文字 / 图片）
2. 自动整理为问题卡片
3. 生成儿童化回答（小朋友版 / 故事版 / 家长提示版）
4. 生成配图
5. 历史回看、搜索、收藏
6. 每周总结
